Theme Info

Theme Info

Available Components

  • CTA Banner — A versatile call-to-action banner with customizable background, content, and button options. Supports both contained and full-width variants.
  • Contact Info — Displays contact information alongside a contact form. Includes email, phone, and location with icons.
  • FAQs — An animated accordion-style FAQ section with expandable questions and answers. Supports custom styling and animations.
  • Features — A grid layout for showcasing features or services with icons, titles, and descriptions. Supports various column layouts.
  • Hero — A full-width hero section with title, description, and call-to-action button. Includes background image support.
  • Inner Hero — A compact hero section for inner pages with title and optional subtitle. Perfect for page headers.
  • Logos — A logo grid for displaying partner or client logos with hover effects and grayscale transitions.
  • Recent Posts — A list of recent posts with titles, excerpts, and featured images. Supports custom styling and animations.
  • Split Panel — A two-column layout combining image and content. Supports left/right image positioning and multiple buttons.
  • Stats — Animated statistics display with counter effects. Supports both grid and side-by-side layouts with optional content.
  • Team Grid — A responsive grid for team members with hover effects and linking to individual profiles.

Configured Collections

  • blog — A blog page with a list of posts and a single post page.
  • team — A team page with a list of team members and a single team member page.
  • legal — A collection of legal documents like Privacy Policy and Terms of Service using Markdown content.

Libraries & Dependencies

  • Astro — Modern static site builder with island architecture ^5.2.5
  • Tailwind CSS — Utility-first CSS framework ^4.0.5
  • AOS — Animate On Scroll library for scroll animations ^2.3.4
  • Lucide Icons — Beautiful & consistent icon pack ^0.473.0
  • Astro Sitemap — Automatic XML sitemap generation for SEO optimization ^3.2.1

Technical Specifications

  • Build System — Static site generation with dynamic islands
  • Performance — Zero JS by default, selective hydration for interactive components
  • Styling — Tailwind CSS with custom theme configuration
  • TypeScript — Full TypeScript support with type safety
  • Asset Handling — Optimized image processing and asset bundling
  • SEO — Configured XML sitemap with weekly changefreq and 404 page exclusion
